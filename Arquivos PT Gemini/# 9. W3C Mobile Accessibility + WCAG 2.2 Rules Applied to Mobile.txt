# 9. Acessibilidade móvel W3C + Regras WCAG 2.2 aplicadas ao mobile

**Contextos W3C:**

* Telas pequenas
* Entrada por toque e voz
* Ambientes com luz/ruído variável

**Princípios WCAG 2.2:**

* Perceptível
* Operável
* Compreensível
* Robusto

**Critérios principais:**

* Alvo mínimo: 48x48dp
* Indicadores de foco visíveis
* Fornecer feedback ao usuário para ações

**Exemplo:**

```kotlin
Button(onClick = { /* ação */ }, modifier = Modifier.size(48.dp)) {
    Text("OK")
}
```
