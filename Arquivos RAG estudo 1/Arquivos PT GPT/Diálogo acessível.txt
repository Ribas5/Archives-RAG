
# 9. Diálogo acessível

```kotlin
AlertDialog(
    onDismissRequest = onCancel,
    title = { Text("Excluir item", modifier = Modifier.semantics { heading() }) },
    text = { Text("Tem certeza que deseja excluir?") },
    confirmButton = {
        TextButton(onClick = onConfirm, modifier = Modifier.semantics {
            contentDescription = "Confirmar exclusão"
        }) {
            Text("Confirmar")
        }
    },
    dismissButton = {
        TextButton(onClick = onCancel, modifier = Modifier.semantics {
            contentDescription = "Cancelar"
        }) {
            Text("Cancelar")
        }
    }
)
```
