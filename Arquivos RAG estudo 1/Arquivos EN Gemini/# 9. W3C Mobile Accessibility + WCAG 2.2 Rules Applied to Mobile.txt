# 9. W3C Mobile Accessibility + WCAG 2.2 Rules Applied to Mobile

**W3C Contexts:**
- Small screens
- Touch and voice input
- Variable light/noise environments

**WCAG 2.2 Principles:**
- Perceivable
- Operable
- Understandable
- Robust

**Key Criteria:**
- Minimum target: 48x48dp
- Visible focus indicators
- Provide user feedback for actions

**Example:**
```kotlin
Button(onClick = { /* action */ }, modifier = Modifier.size(48.dp)) {
    Text("OK")
}
```
